# セキュリティに関する注意事項

## GitHubアップロード前の確認

### ✅ 機密情報の削除

以下の機密情報はコードから削除済みです：

- LINE Messaging APIのチャネルアクセストークン
- LINE Messaging APIのユーザーID
- Googleスプレッドシートの認証情報
- メール通知のSMTP設定

### ⚠️ プライベートリポジトリの使用を推奨

個人利用のみの場合、**必ずプライベートリポジトリ**を作成してください。

### 🔒 Streamlit Secretsでの管理

機密情報はStreamlit CloudのSecrets機能で管理してください。

詳細は`GITHUB_DEPLOY.md`を参照してください。

## コミット前に確認すべきファイル

以下のファイルが`.gitignore`に含まれているか確認：

- `urls.json` - 登録URL（個人情報の可能性）
- `results/` - チェック結果
- `logs/` - ログファイル
- `notification_config.json` - 通知設定（削除済み）
- `*-*.json` - Googleサービスアカウントキー

## 万が一機密情報をコミットしてしまった場合

1. すぐにGitHubでリポジトリを削除
2. 新しいリポジトリを作成
3. 機密情報を変更（トークンやパスワードを再生成）
4. 新しいリポジトリにプッシュ
